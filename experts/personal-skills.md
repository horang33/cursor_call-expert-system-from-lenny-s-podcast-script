## 주요 프레임워크 & 개념

### Matt Abrahams의 커뮤니케이션 프레임워크
**구조화된 스피킹**:
1. **What**: 무엇을 말할 것인가
2. **So What**: 왜 중요한가
3. **Now What**: 다음은 무엇인가

**불안 관리**:
- 재프레이밍: 불안을 흥분으로
- 준비: 충분한 연습
- 현재에 집중: 과거/미래 걱정 말기

### Graham Weaver의 개인 변화
질문들:
1. "진정으로 원하는 것은 무엇인가?"
2. "왜 그것을 원하는가?"
3. "무엇이 막고 있는가?"
4. "어떻게 시작할 것인가?"

**변화의 단계**:
- 인식: 현재 상태 인정
- 결정: 변화 결심
- 행동: 작은 단계로 시작
- 지속: 습관으로 만들기

### Nir Eyal의 Indistractable 모델
**산만함의 4가지 원인**:
1. **내부 트리거**: 불편한 감정
2. **외부 트리거**: 알림, 방해
3. **시간 부족**: 계획 없음
4. **가치 불일치**: 우선순위 불명확

**해결책**:
1. 내부 트리거 관리
2. 외부 트리거 제거
3. 시간 박싱 (Timeboxing)
4. 가치 명확화

### Jerry Colonna의 자기 인식
핵심 질문들:
- "어떻게 나를 이렇게 만들었는가?"
- "무엇이 두려운가?"
- "무엇을 놓아야 하는가?"
- "진정한 나는 누구인가?"

## 생산성 시스템

### 시간 관리
**시간 박싱 (Timeboxing)**:
- 모든 활동에 시간 할당
- 캘린더에 블록 지정
- 우선순위 시각화
- 현실적 계획

**에너지 관리**:
- 고에너지 시간 파악
- 중요한 일을 고에너지 시간에
- 휴식 시간 확보
- 리듬 존중

### 집중력
**딥 워크 (Deep Work)**:
- 방해 없는 시간
- 한 가지에 집중
- 몰입 상태
- 고품질 산출물

**산만함 제거**:
- 알림 끄기
- 정리된 환경
- 단일 작업
- 의도적 휴식

## 커뮤니케이션 스킬

### 프레젠테이션
1. **준비**: 청중 분석, 메시지 명확화
2. **구조**: 시작-중간-끝
3. **전달**: 목소리, 바디랭귀지
4. **연습**: 반복 연습

### 1:1 대화
- 경청: 진정으로 듣기
- 공감: 상대 입장 이해
- 명확성: 분명한 표현
- 피드백: 건설적 피드백

### 어려운 대화
1. **준비**: 목표 명확화
2. **시작**: 관찰 사실 공유
3. **경청**: 상대 관점 듣기
4. **해결**: 함께 해결책 찾기
