## 주요 프레임워크 & 개념

### Brian Balfour의 4가지 성장 핏
1. **Product-Market Fit**: 제품이 시장 니즈를 충족하는가?
2. **Product-Channel Fit**: 제품이 채널에 맞는가?
3. **Channel-Model Fit**: 채널이 비즈니스 모델에 맞는가?
4. **Model-Market Fit**: 비즈니스 모델이 시장에 맞는가?

### Elena Verna의 PLG 프레임워크
- **Acquisition**: 사용자 획득
- **Activation**: 첫 가치 경험 (Aha moment)
- **Engagement**: 지속적 사용
- **Monetization**: 수익화
- **Expansion**: 업셀/크로스셀

### Sean Ellis의 40% 테스트
사용자에게 질문: "이 제품이 없어진다면 어떻게 느끼시겠습니까?"
- 40% 이상이 "매우 실망할 것"이라고 답하면 PMF 달성

### Casey Winters의 리텐션 프레임워크
- **L7/L30**: 지난 7일/30일 중 며칠 사용했는가
- **리텐션 커브**: 시간에 따른 사용자 유지율
- **코호트 분석**: 가입 시기별 리텐션 비교

## 성장 루프 vs 성장 퍼널

### 성장 퍼널 (전통적)
```
인지 → 고려 → 구매 → 유지
(선형적, 외부 투입 필요)
```

### 성장 루프 (현대적)
```
사용자 → 가치 경험 → 공유/추천 → 새 사용자 → ...
(순환적, 자체 강화)
```

## 성장 단계별 전략

### 0 → 1 (PMF 찾기)
- 리텐션에 집중
- 소수의 열성 사용자 확보
- 수동적 방법으로 사용자 획득

### 1 → 10 (채널 찾기)
- 확장 가능한 채널 실험
- 성장 루프 구축
- 초기 팀 구성

### 10 → 100 (최적화)
- 채널별 최적화
- 성장 팀 확대
- 데이터 인프라 구축

### 100+ (지속)
- 새로운 성장 레버 발굴
- 국제화
- 새로운 제품 라인
